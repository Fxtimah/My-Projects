# Fatimah Agboola — Python GitHub Project Pack

This folder contains three Python projects designed to strengthen a graduate software engineering CV.

## 1. Student Management System
Demonstrates:
- Object-Oriented Programming
- SQLite database integration
- CRUD operations
- Input validation
- Report generation

Run:
```bash
python student_management_system.py
```

## 2. Real-Time Weather Dashboard
Demonstrates:
- REST API integration
- JSON parsing
- Error handling
- Modular programming

Install:
```bash
pip install requests
```

Run:
```bash
python weather_dashboard.py
```

## 3. Accessibility Website Auditor
Demonstrates:
- Web scraping
- HTML parsing
- Accessibility awareness
- Automated reporting

Install:
```bash
pip install requests beautifulsoup4
```

Run:
```bash
python accessibility_auditor.py
```

## CV Project Bullets

- Developed a Python-based student management system using object-oriented programming and SQLite database integration to support record management, search functionality, and grade reporting.
- Built a Python weather dashboard integrating external REST APIs to retrieve and display real-time weather data with structured error handling and JSON parsing.
- Created a Python accessibility auditing tool that scans webpages for basic accessibility issues including missing alt text, heading structure issues, and unlabelled form inputs.
